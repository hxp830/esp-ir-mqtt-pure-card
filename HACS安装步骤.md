# HACS 安装 ESP IR MQTT Pure Card 详细步骤

## ✅ GitHub 仓库已成功创建！

**仓库地址**: https://github.com/hxp830/esp-ir-mqtt-pure-card

---

## 📦 方法1：通过 HACS 自定义存储库（推荐）

### 第1步：打开 HACS
1. 在 Home Assistant 左侧菜单找到 **HACS**
2. 点击进入 HACS

### 第2步：进入前端页面
1. 在 HACS 中点击 **Frontend**（前端）标签
2. 或者点击 **前端** 选项卡

### 第3步：打开自定义存储库
1. 点击右上角的 **三个点 ⋮** 菜单按钮
2. 选择 **自定义存储库**（Custom repositories）

### 第4步：添加存储库
1. 在弹出的窗口中：
   - **存储库URL** 输入：
     ```
     https://github.com/hxp830/esp-ir-mqtt-pure-card
     ```
   - **类别** 选择：**Lovelace**
2. 点击 **添加** 按钮

### 第5步：搜索并安装
1. 关闭自定义存储库窗口
2. 在 HACS 前端页面的搜索框中输入：
   ```
   ESP IR MQTT Pure
   ```
3. 找到 **ESP IR MQTT Pure Card**
4. 点击进入
5. 点击右下角的 **下载** 或 **安装** 按钮
6. 等待下载完成

### 第6步：重启 Home Assistant
1. 进入 **设置 > 系统 > 重启**
2. 点击 **重启**
3. 等待重启完成

### 第7步：刷新浏览器
1. 重启完成后，按 **Ctrl+Shift+R**（或 Cmd+Shift+R）
2. 强制刷新浏览器缓存

---

## 📝 方法2：如果 HACS 中找不到

如果执行完上述步骤在 HACS 中还是找不到，可以手动安装：

### 第1步：下载文件
从 GitHub 下载文件：
```
https://github.com/hxp830/esp-ir-mqtt-pure-card/blob/main/esp-ir-mqtt-pure-card.js
```

### 第2步：上传到 Home Assistant
将文件复制到：
```
/config/www/community/esp-ir-mqtt-pure-card/esp-ir-mqtt-pure-card.js
```

### 第3步：添加资源
1. 进入 **设置 > 仪表盘**
2. 点击右上角 **三个点 ⋮**
3. 选择 **资源**
4. 点击 **添加资源**
5. 填入：
   - URL: `/hacsfiles/esp-ir-mqtt-pure-card/esp-ir-mqtt-pure-card.js`
   - 类型: **JavaScript 模块**
6. 点击 **创建**

### 第4步：刷新浏览器
按 **Ctrl+Shift+R** 刷新

---

## 🎨 添加卡片到仪表盘

### 第1步：编辑仪表盘
1. 进入你要添加卡片的仪表盘
2. 点击右上角 **编辑仪表盘** 按钮（铅笔图标）

### 第2步：添加卡片
1. 点击 **添加卡片** 按钮
2. 在搜索框中输入 **ESP IR**
3. 或选择 **手动** 方式

### 第3步：配置卡片
粘贴以下配置：

```yaml
type: custom:esp-ir-mqtt-pure-card
title: 红外遥控器
topic_prefix: newchuangan1/ir
language: zh
```

**重要**：将 `topic_prefix` 改成你的 ESPHome 设备的 MQTT 主题前缀！

### 第4步：保存
1. 点击 **保存** 按钮
2. 点击右上角 **完成** 退出编辑模式

## 🆘 故障排查

### 问题1：HACS 中找不到卡片
**解决**：
1. 确认自定义存储库添加成功
2. 刷新 HACS 页面
3. 等待几分钟后重试

### 问题2：添加卡片时提示找不到类型
**解决**：
1. 确认已在 HACS 中安装
2. 重启 Home Assistant
3. 清除浏览器缓存（Ctrl+Shift+R）

### 问题3：卡片加载但功能不正常
**解决**：
1. 检查 topic_prefix 配置是否正确
2. 确认 MQTT 连接正常
3. 按 F12 查看控制台错误
